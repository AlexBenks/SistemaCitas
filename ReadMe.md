# Sistema de Citas (Consola)

Proyecto personal desarrollado en **Java** como práctica de programación orientada a objetos.

El objetivo es construir un sistema simple que permita registrar y administrar citas desde la consola.

Actualmente el proyecto se encuentra **en desarrollo** como parte de mi aprendizaje.

## Funcionalidades planeadas

* Crear citas
* Ver citas registradas
* Eliminar citas
* Asociar cada cita a un cliente

## Estructura del proyecto

El sistema está basado en dos clases principales:

Cliente

* id
* nombre
* telefono

Cita

* id
* cliente
* fecha
* hora
* motivo

Cada **Cita** está asociada a un **Cliente**.

## Tecnologías utilizadas

* Java
* Programación Orientada a Objetos
* ArrayList para almacenamiento en memoria

## Ejecución del sistema

El sistema funciona mediante un menú en consola:

1. Crear cita
2. Ver citas
3. Eliminar cita
4. Salir

Las citas se almacenan temporalmente mientras el programa está en ejecución.
